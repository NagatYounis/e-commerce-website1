# E-Commerce React App

A modern and responsive e-commerce front-end application built using React.  
The project focuses on product browsing, cart management, and clean UI design.

## Live Demo
e-commerce-website1-iota.vercel.app

## Features
- Display products fetched from a Fake REST API
- Add and remove products from the shopping cart
- Global cart state management using React Context
- Product slider using Swiper

## Tech Stack
- React (Vite)
- React Router DOM for client-side routing
- React Context API for state management
- Framer Motion for animations
- Swiper for product sliders
- React Hot Toast for notifications
- React Icons for UI icons
